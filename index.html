Mis Proyectos

FobiBike - Sistema de Administración de Productos

Un proyecto web que permite la gestión completa de productos para una bicicletería, brindando herramientas de control y organización eficientes: Añadir, editar y eliminar productos de manera sencilla, filtrar productos mediante barra de búsqueda, categorías, tipos, proveedores y stock, consultar información detallada de cada producto y su disponibilidad, optimizado para facilitar la actualización y mantenimiento del inventario. Este proyecto está diseñado para mejorar la eficiencia en la gestión de inventario, combinando funcionalidad, control de datos y facilidad de uso, y se encuentra en constante actualización y mejora.

Info-Play

Este proyecto es una página web interactiva dedicada a ofrecer información completa sobre PlayStation. Incluye secciones sobre la historia de la consola, análisis de juegos y novedades del universo PlayStation. Está desarrollado utilizando HTML y CSS para la estructura y el diseño visual, JavaScript para la interactividad del usuario, y Python para el manejo de datos dinámicos y la integración de contenido, como análisis o estadísticas de juegos. El resultado es una experiencia informativa y atractiva para fanáticos y jugadores interesados en la evolución y el catálogo de PlayStation.
